# [store.youngchief.tk](https://go.youngchief.tk/?url=https://store.youngchief.tk/)

The YoungChief Store!!

To get started:
```bash
git clone https://git.youngchief.tk/gh/youngchief-btw/store.youngchief.tk.git;
cd store.youngchief.tk/;
npm i;
export PORT=[port you want to use]
npm start;
git branch [name of branch];
git checkout [name of branch]
```
